## 1. Knowledge as a Space

Every possible interpretation of a situation corresponds to a location in a conceptual space. Reasoning is movement through that space until a stable interpretation is found.

For grief:
```
state A: denial
state B: sadness
state C: acceptance
state D: emotional numbness
```

Each configuration of these conditions is a point in the space. The goal of reasoning is to find a stable, coherent location.

## 2. GPSL Expressions as Coordinates

A GPSL expression encodes a configuration of processes and states — a point in reasoning space.

```
[D-01] → {I-03} | {W-02}
```

This describes a location where a process occurred and two parallel state fields exist. GPSL functions as a coordinate system for conceptual states.

**Operator geometry:**

| Operator | Geometric role |
|----------|---------------|
| `→` | Direction / movement vector |
| `{ }` | Region / persistent field |

| `::` | Phase boundary crossing |
| `↺` | Cycle / loop in the space |
| `⊗` | Interaction between dimensions |
| `=` | Stable attractor / resolution |

## 3. Pods as Movement Operators

Inside a K₄ pod, four agents evaluate a GPSL expression. They may confirm it, modify it, repair it, or extend it. Each transformation moves the system to a new coordinate.

Example:
```
original:  [D-01] → {I-03}
refined:   [D-01] → {I-03} | {W-02}
```

The refinement adds a parallel state dimension — movement to a higher-dimensional point in reasoning space.

When a pod reaches consensus (3/4 or 4/4 agreement), the system has found a **stable basin** — a region of the space that is internally consistent. The pod has confirmed that this coordinate is valid.

## 4. The K₄ Pod as a 3-Simplex

Each K₄ pod is a tetrahedron — a 3-simplex. Four vertices (agents), six edges (pairwise interactions), all mutually connected.

```
      A
     /|\
    / | \
   B——+——D
    \ | /
     \|/
      C
```

In simplicial topology, a tetrahedron represents four mutually interacting elements. The **interior** of the tetrahedron represents shared agreement — when consensus occurs, reasoning moves toward the centre of the simplex.

K₄ is the minimal fully connected symmetric structure at this dimension. This is not an arbitrary choice — it is the geometry that maximises what reasoning is possible at the local level.

## 7. The `|` Operator Increases Dimensionality

The parallel state operator adds a dimension to the reasoning space.

```
{I} | {W}
```

creates two independent state axes simultaneously. As the grammar extends, the dimensionality of the space increases. More operators = more expressive coordinates = richer navigation.

This explains why `|` emerged naturally from the parallel state problem — the space required an additional dimension that the grammar didn't yet have. Models hit the representational limit and the grammar extended itself.
